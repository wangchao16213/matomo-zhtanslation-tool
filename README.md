## Matomo的汉化脚本
### 说明
当前已经完成了80%大部分插件的汉化(包含收费的插件),里面可能会有翻译不顺畅的地方,欢迎大家通过issue进行提出更恰当的翻译方式


### 使用说明
通过git clone最新的代码,然后打开cpzhfile.py文件将matomoDir变量改成自己的matomo的路径,再命令行进入matomo-zhtanslation-tool目录执行:
```
python cpzhfile.py 
```

然后大功告成
